Welcome to Infinity's Laptop Pool. This pool is dedicated to temporary use. Please request a new laptop if your laptop breaks down permanently.

  • Go to Reserve a Laptop tabto create a new reservation.
  • If you run into any issue, please inform the system administrators via this form.
  • If you need help accessing Room 123, please reach out via information on Contact Us
  • To return a laptop, please bring it back to the designated tray in Room 123. The pool is inventoried every day and the laptop status will be updated by the system admins.
